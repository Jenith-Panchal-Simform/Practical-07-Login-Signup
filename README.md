## Practical 07 Login and Signup Form using React Hook Form and Zod

### Features
- Created a Login and Signup Schema in Zod.
- Created a Login and SignUp Form using React Hook Form.
- Implemented protected routes,where  if user is not logged in navigate him to login Page.
- Store the user details and currently logged in User in local storage

---

## How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jenith-Panchal-Simform/Practical-07-Login-Signup.git
   cd Practical-07-Login-Signup
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development Server**
   ```bash
   npm run dev
   ```
